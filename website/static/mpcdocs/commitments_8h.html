void COMMITMENTS_BC_setup ( csprng *  RNG,
COMMITMENTS_BC_priv_modulus *  m,
octet *  P,
octet *  Q,
octet *  B0,
octet *  ALPHA 
)

RNG is only used to generate the values not explicitly specified. This allows using safe primes P and Q generated externally while still randomly generating B0 and ALPHA. In turn, this allows the user to generate P and Q with ad hoc libraries for the generation of primes instead of the (slow) safe prime generation utility included here.

Parameters
RNGCSPRNG to generate P, Q, B0 and ALPHA
mPrivate modulus to populate
PSafe prime 2p+1. Generated if NULL
QSafe prime 2q+1. Generated if NULL
B0Generator of G_pq as subgroup of Z/PQZ. Generated if NULL
ALPHADLOG exponent for B1 = B0^ALPHA. Generated if NULL
